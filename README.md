# InternHire Portal

InternHire Portal is a responsive web application developed using Bootstrap that simplifies the internship recruitment process for students and organizations. The system allows candidates to browse internship opportunities, submit applications, track application progress, and manage interview call information through an easy-to-use interface.

---

## Features

- Responsive and mobile-friendly user interface
- Internship listing and details page
- Internship application form
- Candidate registration functionality
- Application status tracking
- Interview call management section
- Dashboard for viewing submitted applications
- Modern and intuitive design using Bootstrap
- Easy navigation across different modules

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Purpose of the Project

The objective of this project is to streamline the internship application process by providing a centralized platform where students can apply for internships and track interview opportunities efficiently.
